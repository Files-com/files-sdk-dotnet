# FilesCom.Models.ChatMessage

## Example ChatMessage Object

```
{
  "id": 1,
  "role": "example",
  "content": "example",
  "created_at": "2000-01-01T01:00:00Z"
}
```

* `id` / `Id`  (Nullable<Int64>): Chat Message ID.
* `role` / `Role`  (string): Message role.
* `content` / `Content`  (string): Message content.
* `created_at` / `CreatedAt`  (Nullable<DateTime>): Message creation date/time.
