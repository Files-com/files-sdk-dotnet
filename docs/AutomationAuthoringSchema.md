# FilesCom.Models.AutomationAuthoringSchema

## Example AutomationAuthoringSchema Object

```
{
  "definition_schema": "example",
  "error_families": [
    "example"
  ],
  "nodes": [
    "example"
  ],
  "schema_url": "example"
}
```

* `definition_schema` / `DefinitionSchema`  (object): JSON Schema for active Automation v2 graph definitions.
* `error_families` / `ErrorFamilies`  (object[]): Typed error families accepted by Automation v2 on_error rules.
* `nodes` / `Nodes`  (object[]): Active Automation v2 node authoring metadata.
* `schema_url` / `SchemaUrl`  (string): Stable public URL for the Automation v2 graph definition JSON Schema.
